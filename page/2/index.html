freopen函数

简介

C标准库头文件<stdio.h>中,用于重定向输入输出流.
以指定模式重新指定到另一个文件.

C89函数声明

*FILE freopen(const char filename, const char mode,FILE stream);

C99函数声明

FILE freopen(const char* restrict filename, const char * restrict mode, FILE * restrct stream);

说明

filename: 需要重定向的文件名/文件路径.
mode: 代表访问权限的字符串.
例子：
1. “r”，只读访问
2. “w”，只写访问
3. “a”，追加写入

最小生成树

Kruskal

  1. Q：任一加权无向连通图一定能这么生成生成树吗？试证明.
    A：Kruskal算法可以理解成依次加入可合并子图中两个不连通子子图的边中权值最小者. 考虑加入了i(i < n-1) 条边时,此时由“无向连通”，可得：存在连接两个不相交子图的边. Kruskal算法做的事就是将这样的边中最小的加入子图. 也就是说，我们可以一直加入n-1条边至子图中，且每次加入的边都不会形成回路.
  2. 怎么证明此结果所得为最小生成树？(即验证任意生成树权值和>=此结果所得)
  3. 如果一张图有多棵不同的最小生成树，能否改造此算法，使其能得到所有最小生成树？
