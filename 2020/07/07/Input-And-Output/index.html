freopen函数

简介

C标准库头文件<stdio.h>中,用于重定向输入输出流.
以指定模式重新指定到另一个文件.

C89函数声明

*FILE freopen(const char filename, const char mode,FILE stream);

C99函数声明

FILE freopen(const char* restrict filename, const char * restrict mode, FILE * restrct stream);

说明

filename: 需要重定向的文件名/文件路径.
mode: 代表访问权限的字符串.
例子：
1. “r”，只读访问
2. “w”，只写访问
3. “a”，追加写入
